# おじさんポイポイ！ 企画仕様書

## 概要

「おじさんポイポイ！」は、コンカフェに来るおじさん客を盤面に落として、3つつなげて進化させるスマホ縦画面向けカジュアルパズルです。

テーマは「変でかわいいコンカフェ営業」。
狙いは、短い時間でポコポコ進化する気持ちよさと、神客を作りたい中毒性です。

## コア体験

- おじさんを落とす
- 同じおじさんを3つつなげる
- 進化して高ランク客になる
- キャスト列で会計して売上にする
- 神客が誕生したらフィーバーに入って稼ぐ

プレイヤーの主な判断は次の3つです。

- どこに置けば3つつながるか
- 今は育てるか、会計して空きを作るか
- クレーマーを処理するか、2つ集めて逆転させるか

## 盤面

- 5列 x 5行
- 下から積み上がる
- 1列が満杯でも即終了しない。空いている列があればそちらへ逃がせる
- タイルは時間経過で会計され、売上になる

## 難易度

タイトル画面で3段階を選べます。

| 難易度 | 位置づけ |
| --- | --- |
| かんたん | 基本客だけ。初回レビュー用 |
| ふつう | 会社員、照れ客、ガチ恋が段階的に増える。自動配置までの猶予が約6%短くなる |
| むずかしい | 兄客、疲れ客、迷子、浮気性、同担拒否も段階的に増える。自動配置までの猶予が約12%短くなる |

難易度差は「出てくるおじさんの種類」を主軸にしつつ、上位難易度では自動配置までの猶予も短くします。客の自動会計速度は全難易度で同じにして、放置で勝手に片づきすぎないようにします。

## 進化ルート

| 段階 | 客 | 役割 |
| --- | --- | --- |
| 1 | 薄客 | 基本素材 |
| 2 | 常連 | 中間素材 |
| 3 | 太客 | 高売上素材 |
| 4 | VIP | 神客直前の高価値素材 |
| 5 | 神客 | 誕生したらフィーバー |

同じ客が上下左右で3つ以上つながると、1段階上の客に進化します。進化後にさらに3つつながる場合は連鎖します。

## クレーマー

クレーマーは通常客とは別のリスク素材です。

- 放置すると一定間隔でクレームが増える
- クレームが上限に達すると営業終了
- クレーマーを2つつなげると「苦情仲間」になる
- 苦情仲間は高売上素材として会計できる

狙いは「嫌な駒」ではなく「危険だけどおいしい駒」にすることです。

## キャスト列

キャストは育成の正解ではなく、会計時の倍率・効果です。

| キャスト | 得意 |
| --- | --- |
| みじゅ | 常連 / 太客 x2 |
| ちゃき | クレーマー / 苦情仲間 x3 |
| なの | 薄客 x1.8 |
| いずも | 会計時に空気回復 |
| ゆめ | VIP / 神客 x2.5 |

キャストをタップすると、その列の一番下の客を会計して回収を早めます。

## 神客フィーバー

神客が誕生すると、営業終了ではなく一定時間のフィーバーに入ります。

- フィーバー中の会計売上が2倍
- 自動配置までの猶予が伸びる
- 残り時間が少し回復する
- 空気が大きく回復する

神客は勝利演出ではなく、高スコアを伸ばすためのチャンスタイムです。

## 空気

空気は営業の安定度です。

- 高いほど操作猶予が増える
- 高いほど会計ボーナスが上がる
- 低いと不利になりやすい
- いずも列で会計すると空気が回復する
- クレーマー放置で空気が悪くなる

## 勝敗

勝ち筋:

- 神客フィーバー中に高ランク客を会計する
- 高ランク客を得意キャストで会計して高スコアを狙う
- 連鎖数を伸ばす

営業終了条件:

- 時間切れ
- 全列が満杯で置けない
- クレームが上限に達する

## 初回チュートリアル

最初の3手は、なの列に薄客を3つ落として常連に進化させる誘導を行います。
その後、なのをタップして会計するミニステップを入れ、配置と回収を一度ずつ体験させます。

目的:

- 「3つつなげる」を体験させる
- 進化の気持ちよさを最初に見せる
- キャストタップで会計できることを見せる
- その後の通常営業へ自然につなげる

## リザルト

表示する情報:

- 売上
- 空気
- 最高連鎖
- 進化数
- 次プレイへのヒント
