## これは何？

コンカフェの常連視点を起点にした落ちものパズルのプロトタイプです。  
客をキャスト列に配置し、推し一致、推し被り、VIP指名、クレーマー放置、空気ゲージを見ながら1シフトを回します。

## 触り方

### PC

- 左右キー: 列を選ぶ
- `Space` / `Enter` / `↓`: 客を配置
- `R`: リスタート

### スマホ

- 盤面の列をタップ: その列に配置
- 下部ボタンでも操作可能

## 1プレイで見るポイント

1. 来店中の客が誰推しか見る
2. 選択列の判定を見る
3. 推し一致なら早く帰る
4. 推し被りや指名外は空気が悪くなる
5. 同じ列で連続処理するとボトル / シャンパンが起きる
6. クレーマーを放置すると出禁カウントが進む

## レビュー質問

### 企画

- 「コンカフェをパズルにする」企画として伝わるか
- 客タイプやキャスト類型に納得感があるか
- 不快すぎず、でも丸くなりすぎていないか

### ゲーム性

- 配置判断は面白いか
- 推し一致 / 推し被り / 指名外は理解できるか
- 空気ゲージはプレイに効いていると感じるか
- 1シフトの長さは適切か
- もう1回遊びたいと思えるか

### UI

- 来店中の客と次客は読みやすいか
- 盤面上の客の状態は読みやすいか
- キャスト列ごとのボトル進捗はわかるか
- スマホでも遊べそうか
- 文字量は多すぎないか

### トーン

- テキストの文体は企画に合っているか
- もっとシニカルに寄せるべきか
- もっとカジュアルに寄せるべきか
