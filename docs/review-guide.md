# レビューガイド

## 目的

「おじさんポイポイ！」は、コンカフェに来るおじさん客を5列の盤面に落として、3つつなげて育て、最後にキャスト列で回収するスマホ向けカジュアルパズルです。

## 遊び方

1. 来店中のおじさんを列に落とす
2. 同じおじさんを上下左右で3つつなげる
3. 3つつながると進化する
4. 育ったおじさんはキャスト列で会計して売上にする
5. 神客が誕生したらフィーバー中に高く会計する

進化ルート:

薄客 → 常連 → 太客 → VIP → 神客

クレーマーは放置するとクレームが増えますが、2つつなげると「苦情仲間」になり、高売上の素材になります。

## 操作

- 左右ボタン: 列を移動
- おとす: 来店中のおじさんを配置
- 盤面タップ: その列に配置
- キャストタップ: その列の一番下のおじさんを会計

## 見てほしいこと

- 初見で「3つつなげる」が伝わるか
- 最初の3手チュートリアルで遊び方を理解できるか
- 会計操作に気づけるか
- チュートリアル後の「キャストタップで会計」が自然に押したくなるか
- 神客フィーバーでもう一押し稼ぎたくなるか
- クレーマーが邪魔なだけでなく、逆転素材として面白いか
- 90秒でもう1回遊びたくなるか

## レビュー項目

- ルール理解
- 操作しやすさ
- 盤面の読みやすさ
- 進化したときの気持ちよさ
- クレームによる緊張感
- 神客到達の現実味
- リザルト画面から次プレイへの動機
