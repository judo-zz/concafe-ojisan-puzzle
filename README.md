# コンカフェおじさん落ちものパズル

コンカフェの常連視点を起点にした、ブラウザで遊べる落ちものパズルのプロトタイプです。

## レビュー用ドキュメント

- [企画仕様書](docs/game-design-spec.md)
- [レビューガイド](docs/review-guide.md)

## 遊び方

- 左右キーで列を選択
- `Space` / `Enter` / `↓` で来店中の客を配置
- スマホでは盤面の列をタップして配置
- タイトル画面で `かんたん` / `ふつう` / `むずかしい` を選べます
- 同じ客を3つつなげると進化します: 薄客 → 常連 → 太客 → VIP → 神客
- キャスト列は回収倍率です。育てた客を得意キャストの列で回収すると売上が伸びます
- クレーマーは放置するとクレームが増えますが、2つつなげると「苦情仲間」になって逆転できます
- 神客を作ると会計売上2倍のフィーバータイムに入ります
- `ふつう` と `むずかしい` は出てくるおじさんの種類が増え、自動配置までの猶予も少し短くなります

## 起動

静的HTMLなので、そのまま `index.html` を開けば遊べます。

ローカルサーバーで見る場合:

```sh
python3 -m http.server 8026
```

その後、`http://localhost:8026/` を開いてください。
