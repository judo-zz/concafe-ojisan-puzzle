# コンカフェおじさん落ちものパズル

コンカフェの常連視点を起点にした、ブラウザで遊べる落ちものパズルのプロトタイプです。

## レビュー用ドキュメント

- [企画仕様書](docs/game-design-spec.md)
- [レビューガイド](docs/review-guide.md)

## 遊び方

- 左右キーで列を選択
- `Space` / `Enter` / `↓` で来店中の客を配置
- スマホでは盤面の列をタップして配置
- 推しキャストの列に置くと早く満足して帰ります
- 推し被り、指名外VIP、クレーマー放置は空気を悪化させます
- 同じキャスト列で客を連続処理するとボトル、シャンパンコールが発生します

## 起動

静的HTMLなので、そのまま `index.html` を開けば遊べます。

ローカルサーバーで見る場合:

```sh
python3 -m http.server 8026
```

その後、`http://localhost:8026/` を開いてください。
